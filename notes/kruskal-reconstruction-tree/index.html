In the last ICPC training, I encountered a problem about Kruskal Reconstruction Tree in which I have never learnt before, so I decided to write a note about it.

To understand Kruskal Reconstruction Tree, you should first understand the Kruskal algorithm for minimum spanning trees.

What is Kruskal Reconstruction Tree?

In Kruskal algorithm, we do a greedy algorithm for constructing a minimum spanning tree. We sort the edges by their weights and add the edges one by one, if the edge does not form a cycle with the edges we have added before.

Note

Observe that the minimum spanning tree gives the edges that form the paths with the smallest bottleneck weight between any two vertices.
